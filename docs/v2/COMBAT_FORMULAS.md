# 호이상사 V2 전투 공식 기준

이 문서는 V2 스탯창과 실제 전투가 공유할 1차 전투 계산 기준이다.

## 스탯 대응

| 기존 표기 | V2 표기 |
| --- | --- |
| STR | 맷집 (`grit`) |
| DEX | 처리속도 (`processingSpeed`) |
| INT | 업무지식 (`workKnowledge`) |
| LUK | 눈치 (`awareness`) |

## 무기상수

| 무기 | 상수 |
| --- | ---: |
| 한손검 | 4.0 |
| 두손검 | 4.6 |
| 한손도끼, 한손둔기 | 4.4 |
| 두손도끼, 두손둔기 | 4.8 |
| 창, 폴암 | 5.0 |
| 활 | 3.4 |
| 석궁 | 3.6 |
| 아대, 단검 | 3.6 |
| 너클 | 4.8 |
| 건 | 3.6 |

`공격력`은 무기, 장갑, 탄환, 화살, 버프 등 모든 공격력의 합이다.

## 물리 스탯 공격력

- 전사/너클 해적: `(STR × 무기상수 + DEX) × 공격력 ÷ 100`
- 궁수/건 해적: `(DEX × 무기상수 + STR) × 공격력 ÷ 100`
- 도적: `(LUK × 무기상수 + STR + DEX) × 공격력 ÷ 100`
- 최소 스탯 공격력: `최대 스탯 공격력 × 숙련도`
- 공격력 1당 주스탯 환산: `스탯공식 분자 ÷ (공격력 + 1) ÷ 무기상수`

숙련도는 내부적으로 `0~1`을 사용하며, `70`처럼 입력하면 `0.7`로 정규화한다.

## 마법 스킬 피해

- 최대: `{(마력² ÷ 1000 + 마력) ÷ 30 + INT ÷ 200} × 스킬 공격력`
- 최소: `{(마력² ÷ 1000 + 마력 × 0.9 × 숙련도) ÷ 30 + INT ÷ 200} × 스킬 공격력`

마력 500, INT 400, 스킬 공격력 100이면 방어 계산 전 최대 피해는 2,700이다.

## 명중률과 회피율

명중률:

- 초보자/전사/법사: `DEX × 0.8 + LUK × 0.5 + 추가 명중률`
- 인파이터: `DEX × 0.9 + LUK × 0.3 + 추가 명중률`
- 궁수/도적/해적 1차/건슬링거: `DEX × 0.6 + LUK × 0.3 + 추가 명중률`

회피율:

- 초보자/전사/법사/궁수/도적/해적 1차: `DEX × 0.25 + LUK × 0.5 + 추가 회피율`
- 인파이터: `DEX × 1.5 + LUK × 0.5 + 추가 회피율`
- 건슬링거: `DEX × 0.125 + LUK × 0.5 + 추가 회피율`

필요 명중률:

`{55 + 2 × max(0, 몬스터 레벨 - 캐릭터 레벨)} × 몬스터 회피율 ÷ 15`

실제 적중률:

- 현재 명중률이 필요 명중률 이상이면 적중률 100%
- 부족하면 `max(5%, (현재 명중률 ÷ 필요 명중률)^1.25)`
- MISS 확률은 `1 - 적중률`

예시: 요구치의 75%는 약 69.8% 적중, 50%는 약 42.0% 적중, 명중률이 0이어도 최소 5% 적중한다.

## 몬스터 방어 계산

`D = max(0, 몬스터 레벨 - 캐릭터 레벨)`

물리 공격은 방어력을 먼저 빼고 스킬 퍼센트를 적용한다.

- 최대: `최대 스공 × (1 - 0.01 × D) - 물리방어력 × 0.5`
- 최소: `최소 스공 × (1 - 0.01 × D) - 물리방어력 × 0.6`

마법 공격은 스킬 피해를 먼저 계산하고 방어력을 적용한다.

- 최대: `최대 스킬 피해 - 마법방어력 × 0.5 × (1 + 0.01 × D)`
- 최소: `최소 스킬 피해 - 마법방어력 × 0.6 × (1 + 0.01 × D)`

최종 피해는 0 미만으로 내려가지 않는다.
