## 구현 범위

- 전체 41개 맵 중 29개 맵에 NPC 배치: 약 71%

- NPC 클릭 시 대화·목표·진행도·보상·수락/완료 버튼 표시
- 별도 `퀘스트` 메뉴에서 수락한 퀘스트와 진행 상황 확인
