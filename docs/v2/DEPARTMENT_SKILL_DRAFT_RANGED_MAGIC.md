> 기준일: 2026-07-03

## 1. 문서 규칙

- `p.`는 패시브, `q.`는 퀘스트 해금 스킬이다.
- 별도 표기가 없는 스킬은 액티브 스킬이다.
- 수치는 `1레벨 → 마스터 레벨` 순서다. 한 값만 있으면 전 레벨 고정이다.
- 사거리는 V2 내부 단위의 기획값이다. 현재 초근접 기본공격 `100`을 기준으로 잡았다.
- `범위`는 시전자 중심 또는 타격 지점 중심 반경, `이동`은 이동 거리다.
- 삭제된 원본 스킬은 제외했다.
- 원본 문서에 이름만 남아 수치가 없는 스킬은 `보류`로 표시했다.
- 원본의 즉사, 경험치, 드롭률, 메소 소모 효과는 V2 경제와 보스 밸런스에 맞춰 출시 전에 다시 확정한다.
- 스킬 레벨 중간값은 구현 시 기존 V2 보간 규칙으로 자동 계산한다.

## 35.1. 공통
