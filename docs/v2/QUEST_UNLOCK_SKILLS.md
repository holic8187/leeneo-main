# 퀘스트 해금 스킬 목록
(퀘스트 필요 여부와 관계없이 스킬 마스터리 북 20,30이 필요한 스킬들-전사는 개별로 적어둠. 샤프아이즈, 쇼다운, 마나 리플렉션, 빅뱅, 트리플스로우, 쇼다운, 페이크, 부메랑스텝, 메테오, 인피니티, 체인라이트닝, 블리자드, 홀리실드, 제네시스)
현재 코드에서 `quest: true`로 지정된 스킬을 부서별로 정리한 목록이다. 모두 4차 스킬이며, 실제 투자 전 별도의 스킬 해금 퀘스트 완료가 필요하다.
++퀘스트를 깨면 10까지 찍을수 있으며, 연계퀘스트가 필요함. 연계퀘스트를 깰때마다 마스터 레벨이 10->20->30으로 해금되는 시스템. 20 30 스킬 마스터리북이 존재하는 경우는 스킬북을 통해 해금. 스킬 마스터리북은 20북의 경우 90%, 30북의 경우 70% 확률로 해금에 성공함. 해금에 실패하는 경우 다음 시도에 1%의 추가 성공 확률이 누적으로 더해짐.
따라서 20, 30북이 필요한 스킬은 따로 스킬북을 만들어둬야함. 몬스터에게서 드랍되게 해둬야하고, 드랍되는 몬스터는 추후 지정 예정

전직원 역량강화-기본 10 찍을수 있으며, 스킬북으로 20까지 해금가능
